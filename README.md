# Angular Foundation & Application Architecture – Exercises

This folder contains the exercises I completed while following a course:

**Angular Foundation & Application Architecture**

## Purpose

The goal of these exercises is to practice and reinforce core Angular concepts covered in the course through hands-on coding.

## Topics Covered

- Angular project structure
- Components and templates
- Data binding (interpolation, property, event binding)
- Directives
- Pipes
- Services and dependency injection
- Routing and navigation
- Forms basics
- Application architecture best practices

## Notes

These exercises are part of my learning journey in Angular. They are not full projects but practice implementations of individual concepts.
