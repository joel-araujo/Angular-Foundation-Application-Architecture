# Angular Foundation & Application Architecture – Exercises

This repository contains the exercises and practice implementations completed while following the course:

**Angular Foundation & Application Architecture**

The purpose of this repository is to build a strong foundation in Angular development by exploring core concepts, application structure, and architectural principles required to create maintainable and scalable applications.

---

# Overview

This repository represents my learning journey through Angular fundamentals and application architecture.

The exercises are organized progressively, starting with basic Angular concepts and moving toward more complete application examples.

Each folder focuses on specific Angular features and demonstrates how different parts of the Angular framework work together.

These are **learning exercises and practice implementations**, not production-ready applications. The main objective is to understand Angular concepts through hands-on development.

---

# Learning Goals

The main goals of these exercises are:

- Understand Angular project structure
- Learn component-based application development
- Understand communication between components and templates
- Practice Angular data binding
- Work with directives and pipes
- Learn service creation and dependency injection
- Understand routing and navigation
- Build forms and handle user input
- Apply Angular application architecture principles
